# Movie Trailer Site
<h3>Server-side code to store a list of movies and display their trailers.</h3>

## Install

CLone or download from [here](https://github.com/SyedHimaayath/movie-trailer-site.git)

## Run

use terminal 
```
	python movie_list.py
```
